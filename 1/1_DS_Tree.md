# Data Structures Classification (ASCII Tree)

```text
Data Structures
│
├── 1. Primitive Data Structures
│   │
│   ├── Integer (int)
│   ├── Float (float)
│   ├── Double (double)
│   ├── Character (char)
│   ├── Boolean (bool)
│   └── Pointer
│
└── 2. Non-Primitive Data Structures
    │
    ├── A. Linear Data Structures
    │   │
    │   ├── Array
    │   ├── Linked List
    │   │   ├── Singly Linked List
    │   │   ├── Doubly Linked List
    │   │   └── Circular Linked List
    │   │
    │   ├── Stack
    │   └── Queue
    │       ├── Simple Queue
    │       ├── Circular Queue
    │       ├── Priority Queue
    │       └── Deque
    │
    └── B. Non-Linear Data Structures
        │
        ├── Trees
        │   ├── General Tree
        │   ├── Binary Tree
        │   ├── Binary Search Tree (BST)
        │   ├── AVL Tree
        │   ├── Heap
        │   ├── B-Tree
        │   └── Trie
        │
        └── Graphs
            ├── Directed Graph
            ├── Undirected Graph
            ├── Weighted Graph
            └── Unweighted Graph
```

# Simplified Version (Most Colleges)

```text
                     DATA STRUCTURES
                            │
          ┌─────────────────┴─────────────────┐
          │                                   │
    Primitive                          Non-Primitive
          │                                   │
   int, char, float,                  ┌──────┴──────┐
   double, bool, etc.                 │             │
                                   Linear      Non-Linear
                                      │             │
                     ┌────────────────┼────────┐    ├────────────┐
                     │                │        │    │            │
                  Array          Linked List Stack Queue      Tree      Graph
```

### Exam Shortcut

```
Data Structures
        │
 ┌──────┴──────┐
 │             │
Primitive  Non-Primitive
               │
       ┌───────┴────────┐
       │                │
    Linear         Non-Linear
       │                │
 Array LL Stack Queue  Tree Graph
```

This 4-level hierarchy is the one most commonly expected in Data Structures exams.
